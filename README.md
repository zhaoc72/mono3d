# Mono3D: Single-Image and Video 3D Reconstruction

A comprehensive framework for 3D object reconstruction from monocular images and videos, leveraging state-of-the-art vision foundation models (DINOv3, SAM 2, Depth Anything) and 3D Gaussian Splatting.

## Features

- 🎯 **Single-Image Reconstruction**: Reconstruct 3D objects from a single RGB image
- 🎥 **Video Reconstruction**: Multi-view reconstruction from video sequences
- 🧠 **Foundation Models**: Integrates DINOv3, SAM 2, and Depth Anything
- 🌟 **3D Gaussian Splatting**: High-quality 3D representation with real-time rendering
- 📊 **Shape Priors**: Explicit and implicit shape priors for better reconstruction
- 🚀 **Modular Design**: Easy to extend and customize
- 📈 **Comprehensive Evaluation**: Built-in metrics and visualization tools
